# 《守底线，敢担当》即梦直投版 V3.0（复制即用）

**使用方法（3步）：**
1. 每个镜头先复制【文生图】那段 → 即梦"图片生成"里粘贴，选 21:9 或 16:9，生成首帧图；
2. 再把首帧图导入即梦"视频生成-图生视频"，复制【图生视频】那段粘贴，时长按标注选 5秒 或 10秒；
3. 台词/旁白全部后期配音（剪映配音即可），画面里人物只做说话动作，不追求口型。

**保持人脸一致的关键：** 镜头1生成满意后，把小王和周工的脸截出来，之后每个含他们的镜头都用即梦的"参考图/角色一致性"上传这张脸，提示词不用改。

角色外观已写进每条提示词里，无需你再补：
- 小王：23岁年轻男性，清爽黑色短发，白衬衫，胸前挂蓝色工牌
- 王工（=长大的小王）：28岁男性，浅蓝衬衫袖口挽起，胸前挂蓝色工牌
- 周工：40岁男性，短发两鬓微白，深蓝色polo衫，胸前挂蓝色工牌

## 剪辑清单
- 18个镜头按序拼接，总时长约2分20秒
- 配音：旁白用沉稳男声（剪映"解说男声"类），台词按角色分配

- 音效点：警报声（镜头3-8）、电话铃（镜头6、15）、键盘声、落版音效
